# AI Coding Tool Adoption: Industry Analysis (2016-2026)

A research report analyzing the adoption trajectory of AI-assisted coding tools across the software development industry, from early ML-enhanced autocomplete (2018) through the generative LLM wave (2021+) to agentic coding systems (2025-2026).

## What's Inside

The report in [`docs/deep-research-report.md`](docs/deep-research-report.md) covers:

- **Adoption trends** — Stack Overflow, DORA, HackerRank, JetBrains, and GitHub survey data showing the rise from ~44% usage (2023) to ~79% (2025)
- **Tool-by-tool penetration** — Copilot, ChatGPT, Claude Code, Cursor, Windsurf, Codex, Junie, and Tabnine with the best available public metrics for each
- **Assistive vs. agentic split** — Why ~79% use AI assistance but only ~31% use AI agents, and the trust/verification dynamics behind this gap
- **Segmentation** — Adoption patterns by career stage, role specialization, company size, geography, and enterprise policy posture
- **Productivity and trust** — Controlled experiments, self-reported gains, and the central paradox of heavy usage with low trust (46% distrust vs. 33% trust)
- **Data gaps** — Where measurement is weakest and practical methods to close those gaps
- **Product timeline** — Key milestones from IntelliCode (2018) through Copilot's 4.7M paid subscribers (Jan 2026)
